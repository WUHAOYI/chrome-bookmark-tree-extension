# Bookmark Folder Picker

点击扩展图标弹出完整的书签目录树，选择一个目录后保存当前页面到该目录。

## 安装（开发者模式）

1. clone该项目到本地目录
2. 打开 `chrome://extensions/`
3. 右上角开启“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择本目录 `chrome-bookmark-tree-extension`
6. 在扩展列表中将本扩展“固定”到工具栏

## 使用

1. 打开你想收藏的网页
2. 点击扩展图标
3. 按需修改书签名称
4. 在目录树中点选目标文件夹
5. 点击“保存”

功能点：

- 显示完整书签文件夹目录树（支持展开/折叠）
- 保存前可编辑书签名称
- 搜索文件夹名称
- 新建/删除书签文件夹
- 记忆上次使用的保存目录（下次默认选中）

## 权限说明

- `bookmarks`：读取/创建书签
- `tabs`：获取当前活动标签页的标题和 URL
- `storage`：保存“上次选择的目录”
