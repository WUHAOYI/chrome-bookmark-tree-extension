# Bookmark Folder Picker

点击扩展图标（或快捷键）弹出完整的书签目录树，选择一个目录后保存当前页面到该目录。

## 重要限制

Chrome 扩展无法拦截或替换浏览器内置的地址栏“星形”按钮点击行为，因此无法做到“点击内置星形按钮就弹出自定义目录树”。本扩展提供一个等价的入口：

- 固定到工具栏后，点击扩展图标打开目录树选择器
- 或使用快捷键打开（默认：`Cmd+Shift+D` / `Ctrl+Shift+D`）

## 安装（开发者模式）

1. 打开 `chrome://extensions/`
2. 右上角开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本目录 `chrome-bookmark-tree-extension`
5. 在扩展列表中将本扩展“固定”到工具栏

## 使用

1. 打开你想收藏的网页
2. 点击扩展图标（或按快捷键）
3. 在目录树中点选目标文件夹
4. 点击“保存”

功能点：

- 显示完整书签文件夹目录树（支持展开/折叠）
- 搜索文件夹名称
- 新建/删除书签文件夹
- 记忆上次使用的保存目录（下次默认选中）

## 权限说明

- `bookmarks`：读取/创建书签
- `tabs`：获取当前活动标签页的标题和 URL
- `storage`：保存“上次选择的目录”
